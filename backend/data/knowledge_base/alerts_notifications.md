# Alerts & Notifications

## Setting up alerts
Go to Alerts & Notifications > Manage Alerts. Choose the account, alert type, and delivery channel (email or SMS).

## Available alert types
- **Low balance alert**: Triggered when balance drops below your set threshold.
- **Large debit/credit alert**: Triggered when a transaction exceeds a specified amount.
- **Failed transaction alert**: Notified when a payment or transfer fails.
- **Pending authorisation alert**: Notified when a transaction is waiting for Checker approval.
- **Login alert**: Notified each time a user logs in to Velocity.
- **New user alert**: Notified when a new user is added to the account.

## Delivery channels
- Email: Up to 5 recipients per alert.
- SMS: Up to 2 mobile numbers per alert.

## Editing or deleting alerts
Go to Alerts & Notifications > Manage Alerts > select alert > Edit or Delete.

## Bulk alert setup
Admins can configure alerts for multiple accounts simultaneously under Alerts > Bulk Setup.

## Alert delays
Alerts are sent in near real-time but may be delayed up to 5 minutes depending on network conditions.
