# Payroll

## Payroll via Velocity
Submit bulk salary disbursements using GIRO payroll or FAST payroll (real-time for FAST-enabled banks).

## Uploading payroll file
Go to Payroll > Upload Payroll File. Supported formats: OCBC payroll template (CSV/XLS) or IBG (Interbank GIRO) format.

## Payroll template
Download the standard template from Payroll > Download Template. Required fields: employee name, bank code, account number, amount, payment reference.

## CPF submission
CPF contributions can be paid directly via Velocity > Payroll > CPF Payment. Upload the CPF file exported from your payroll software.

## Cut-off time
- GIRO payroll: Submit by 10:30 PM SGT for next-business-day credit.
- FAST payroll: Credited within minutes, 24/7 (for FAST-participating banks).

## Payroll approval
Bulk payroll requires Maker-Checker authorisation. After the Maker uploads and submits, the Checker must approve before the payment is processed.

## Scheduling
Payroll can be scheduled up to 30 days in advance. View scheduled payroll runs under Payroll > Scheduled.
