# Report Generation

## Available reports
- **Account Summary Report**: Balances across all accounts at a point in time.
- **Transaction Report**: Detailed debit/credit listing with filters.
- **Payroll Report**: Salary disbursement history.
- **Cash Flow Report**: Inflow/outflow summary by account or period.
- **SWIFT MT940 Report**: For ERP/accounting system integration.

## Generating a report
Go to Reports > select report type > set parameters (account, date range, format) > Generate.

## Export formats
PDF and CSV. CSV exports are suitable for import into accounting software (QuickBooks, Xero, SAP, etc.).

## Scheduled reports
Reports can be auto-generated and emailed on a daily, weekly, or monthly basis. Set up under Reports > Scheduled Reports.

## Report retention
Generated reports are stored in Velocity for 30 days. Download them before expiry.

## Custom reports
For bespoke reporting requirements, contact your OCBC Relationship Manager or submit a request via Service Requests.
