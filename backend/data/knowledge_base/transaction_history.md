# Transaction History

## Viewing transactions
Go to Accounts > Transaction History. Select the account and date range (up to 12 months online).

## Search and filter
Filter by date range, transaction type (debit/credit), amount range, or reference number. Use the search bar to look up specific descriptions.

## Export options
Download transactions as CSV or PDF. Select the date range and format, then click Export.

## Transaction statuses
- **Completed**: Successfully posted to the account.
- **Pending**: Authorisation pending (Maker-Checker) or processing by the bank.
- **Rejected**: Transaction was declined or failed validation.

## Older records
For transactions older than 12 months, request a historical statement via Account Statement or contact OCBC Business Customer Service.

## Reconciliation
The exported CSV includes columns for date, description, reference, debit, credit, and running balance — compatible with most accounting software.
