# Deleting / Deactivating a User

## Steps to deactivate a user
Go to Administration > User Management, select the user, and click Deactivate. The user immediately loses login access.

## Reactivation
Reactivating a deactivated user requires Admin action. Go to Administration > User Management, select the user, and click Reactivate.

## Permanent removal
To permanently remove a user, select the user and click Delete. This action cannot be undone and removes all associated access and history links.

## Maker-Checker requirement
All user deactivations and deletions require a second Admin to approve (Maker-Checker) for security.

## Audit trail
All user removal actions are logged under Administration > Audit Trail.
